# 🎯 快速开始 - 3 种方法让手机访问黄历网站

## ⚡ 方法 1：在线部署（最推荐）

### 使用 Netlify Drop - 无需注册，3 分钟搞定！

1. 打开浏览器访问：**https://app.netlify.com/drop**
2. 将 `Huangli` 文件夹拖拽到网页中
3. 等待上传完成，获得链接（例如：`https://xxx.netlify.app`）
4. 在手机浏览器打开这个链接 ✅

**优点：**
- ✅ 全球任何地方都能访问
- ✅ 自动 HTTPS 加密
- ✅ 永久免费
- ✅ 无需安装任何软件

---

## 🏠 方法 2：本地网络（快速测试）

### 双击运行批处理文件

1. 双击 `启动服务器.bat` 文件
2. 查看显示的手机访问地址（例如：`http://192.168.1.86:8000`）
3. 确保手机和电脑连接同一个 WiFi
4. 在手机浏览器输入该地址 ✅

**优点：**
- ✅ 快速启动
- ✅ 无需上传
- ✅ 适合临时测试

**注意：**
- ⚠️ 仅限同一 WiFi 网络
- ⚠️ 需要电脑一直开着

---

## 🌐 方法 3：GitHub Pages（长期使用）

### 适合长期稳定使用

1. 注册 GitHub 账号：https://github.com
2. 创建新仓库（Public）
3. 上传 `index.html`、`index.css`、`script.js`
4. 在 Settings → Pages 启用 GitHub Pages
5. 访问：`https://你的用户名.github.io/仓库名/` ✅

**优点：**
- ✅ 完全免费
- ✅ 稳定可靠
- ✅ 支持自定义域名
- ✅ 版本控制
