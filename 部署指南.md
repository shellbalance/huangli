# 黄历网站部署指南

本指南提供多种方式将黄历网站部署到服务器，让你能在手机上访问。

## 🚀 方案一：Vercel 部署（推荐）

**优点：** 完全免费、全球访问、自动 HTTPS、CDN 加速

### 步骤：

1. **访问 Vercel 官网**
   - 打开浏览器访问：https://vercel.com
   - 点击 "Sign Up" 注册账号（可以使用 GitHub、GitLab 或 Email）

2. **安装 Vercel CLI**
   ```powershell
   # 首先安装 Node.js（如果还没安装）
   # 访问 https://nodejs.org 下载并安装 LTS 版本
   
   # 安装完 Node.js 后，安装 Vercel CLI
   npm install -g vercel
   ```

3. **部署网站**
   ```powershell
   # 进入项目目录
   cd c:\good\Huangli
   
   # 登录 Vercel
   vercel login
   
   # 部署（首次部署）
   vercel
   
   # 按照提示操作：
   # - Set up and deploy? Yes
   # - Which scope? 选择你的账号
   # - Link to existing project? No
   # - What's your project's name? huangli（或其他名称）
   # - In which directory is your code located? ./
   # - Want to override the settings? No
   ```

4. **获取访问链接**
   - 部署完成后，Vercel 会提供一个链接，例如：`https://huangli-xxx.vercel.app`
   - 在手机浏览器中打开这个链接即可访问

---

## 🌐 方案二：GitHub Pages 部署

**优点：** 完全免费、稳定可靠

### 步骤：

1. **创建 GitHub 账号**
   - 访问 https://github.com 注册账号

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名称：`huangli`
   - 设置为 Public
   - 点击 "Create repository"

3. **上传文件**
   - 在仓库页面点击 "uploading an existing file"
   - 将 `index.html`、`index.css`、`script.js` 三个文件拖拽上传
   - 点击 "Commit changes"

4. **启用 GitHub Pages**
   - 进入仓库的 Settings → Pages
   - Source 选择 "main" 分支
   - 点击 "Save"
   - 等待几分钟后，访问：`https://你的用户名.github.io/huangli/`

---

## 📱 方案三：本地网络部署（仅限同一 WiFi）

**优点：** 快速测试、无需注册账号

### 使用 Python HTTP 服务器：

1. **安装 Python**
   - 访问 https://www.python.org/downloads/
   - 下载并安装 Python 3.x

2. **启动服务器**
   ```powershell
   cd c:\good\Huangli
   python -m http.server 8000
   ```

3. **获取本机 IP 地址**
   ```powershell
   ipconfig
   # 找到 "IPv4 地址"，例如：192.168.1.100
   ```

4. **在手机上访问**
   - 确保手机和电脑在同一 WiFi 网络
   - 在手机浏览器中访问：`http://192.168.1.100:8000`

### 使用 Node.js HTTP 服务器：

1. **安装 Node.js**
   - 访问 https://nodejs.org
   - 下载并安装 LTS 版本

2. **启动服务器**
   ```powershell
   cd c:\good\Huangli
   npx serve -l 8000
   ```

3. **在手机上访问**
   - 查看终端显示的网络地址
   - 在手机浏览器中访问显示的地址

## 🔥 方案四：使用免费托管平台

### Netlify Drop（最简单）

1. **访问 Netlify Drop**
   - 打开 https://app.netlify.com/drop

2. **拖拽文件夹**
   - 直接将 `c:\good\Huangli` 文件夹拖拽到网页中
   - 等待上传完成

3. **获取链接**
   - 上传完成后会自动生成一个链接
   - 在手机上访问这个链接
