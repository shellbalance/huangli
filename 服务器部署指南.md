## 🌟 方法二：Vercel 部署（推荐）

### 为什么选择 Vercel？
- ✅ 全球 CDN 加速，访问速度快
- ✅ 自动 HTTPS 证书
- ✅ 无限带宽
- ✅ 支持自定义域名
- ✅ 自动部署更新

### 快速部署步骤：

#### 方式 A：使用自动脚本（推荐）

1. **右键点击** `deploy-vercel.ps1`
2. **选择** "使用 PowerShell 运行"
3. **按照提示操作**：
   - 首次使用需要登录 Vercel（会自动打开浏览器）
   - 选择登录方式（GitHub/GitLab/Email）
   - 等待部署完成

#### 方式 B：手动部署

```powershell
# 1. 安装 Vercel CLI（如果未安装）
npm install -g vercel

# 2. 登录 Vercel
vercel login

# 3. 进入项目目录
cd c:\good\Huangli

# 4. 部署到生产环境
vercel --prod
```

### 部署后：

- 你会获得一个链接，例如：`https://huangli-xxx.vercel.app`
- 在手机浏览器中打开这个链接即可访问
- 每次修改代码后，重新运行 `vercel --prod` 即可更新

## 🎨 方法三：Netlify 部署

### 为什么选择 Netlify？
- ✅ 界面友好，操作简单
- ✅ 拖拽上传，无需命令行
- ✅ 自动 HTTPS 证书
- ✅ 免费 CDN 加速

### 快速部署步骤：

#### 方式 A：使用自动脚本

1. **右键点击** `deploy-netlify.ps1`
2. **选择** "使用 PowerShell 运行"
3. **按照提示操作**

#### 方式 B：Netlify Drop（最简单）

1. **打开浏览器**，访问：https://app.netlify.com/drop

2. **拖拽文件夹**：
   - 将整个 `Huangli` 文件夹拖到网页中
   - 或者选择 `index.html`、`index.css`、`script.js` 三个文件

3. **等待上传**（几秒钟）

4. **获取链接**：
   - 例如：`https://random-name-123456.netlify.app`
   - 在手机上访问这个链接

#### 方式 C：使用 CLI

```powershell
# 1. 安装 Netlify CLI
npm install -g netlify-cli

# 2. 登录 Netlify
netlify login

# 3. 部署
cd c:\good\Huangli
netlify deploy --prod --dir .
```

## 📱 方法四：GitHub Pages 部署

### 适合场景：
- 长期稳定使用
- 需要版本控制
- 想要免费托管

### 部署步骤：

1. **创建 GitHub 账号**
   - 访问：https://github.com
   - 点击 "Sign up" 注册

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名称：`huangli`
   - 设置为 Public
   - 点击 "Create repository"

3. **上传文件**
   - 点击 "uploading an existing file"
   - 拖拽上传：`index.html`、`index.css`、`script.js`
   - 点击 "Commit changes"

4. **启用 GitHub Pages**
   - 进入 Settings → Pages
   - Source 选择 "main" 分支
   - 点击 "Save"

5. **访问网站**
   - 等待 1-2 分钟
   - 访问：`https://你的用户名.github.io/huangli/`

## 🏠 方法五：本地网络部署

### 适合场景：
- 快速测试
- 不想注册账号
- 只在家里使用

### 使用现有脚本：

**双击运行** `启动服务器.bat`，脚本会：
- 自动检测 Python 或 Node.js
- 启动本地服务器
- 显示访问地址

### 手动部署：

#### 使用 Python：
```powershell
cd c:\good\Huangli
python -m http.server 8000
```

#### 使用 Node.js：
```powershell
cd c:\good\Huangli
npx serve -l 8000
```

### 获取访问地址：

1. **查看电脑 IP**：
   ```powershell
   ipconfig
   ```
   找到 "IPv4 地址"，例如：`192.168.1.86`

2. **在手机上访问**：
   - 确保手机和电脑在同一 WiFi
   - 访问：`http://192.168.1.86:8000`

## 🔧 常见问题解决

### 问题 1：PowerShell 脚本无法运行

**解决方法**：
```powershell
# 以管理员身份运行 PowerShell，执行：
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 问题 2：npm 命令不存在

**解决方法**：
1. 访问 https://nodejs.org
2. 下载并安装 LTS 版本
3. 重启终端

### 问题 3：本地网络无法访问

**检查清单**：
- [ ] 手机和电脑在同一 WiFi
- [ ] 防火墙允许端口访问
- [ ] IP 地址正确
- [ ] 服务器正在运行

**Windows 防火墙设置**：
```powershell
# 允许端口 8000
netsh advfirewall firewall add rule name="HTTP Server" dir=in action=allow protocol=TCP localport=8000
```

### 问题 4：部署后无法访问

**检查步骤**：
1. 确认部署成功（查看终端输出）
2. 等待 1-2 分钟（DNS 传播）
3. 清除浏览器缓存
4. 尝试使用无痕模式访问
